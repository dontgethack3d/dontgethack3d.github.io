Password

Learn how to create and store passwords to keep your data safe.

Intro

The Internet is where we find everything.
We use many sites, many of which we have our accounts.
Indeed, none of us wants anyone but us to have access to our account on Facebook, for example. Therefore, we should create strong passwords according to the rules below.

Creating Passwords

Rule 1

The password should contain:

  • 🔐 minimum 8 characters
  • 🔠 upper and 🔡 lower case
  • 🔢 digits
  • 🔣 special characters

Rule 2

The password should not be easy to guess.
It should not contain, for example, your name, date of birth or other information related to you.
The best passwords are those that contain random characters.

Keep Your Passwords Safe

If you already have a strong password, you need to remember that there is more to it than that. You need to follow a few rules to store passwords.

Do not save the password in files.

Someone accessing your computer can get access to all your data.

Don't save password in browser!

Storing passwords in the browser is a dangerous solution.
The browser is the obvious place to save passwords, plus it doesn't encrypt them, making it possible to access them after downloading a virus, for example.
A better solution is to use an application like Bitwarden or 1password for this.

Don't enter passwords on untrusted sites.

Only enter passwords on sites you trust. Always check that the site address is correct.
Remember that the 🔒 icon does not mean that a scammer did not create the site.

Use Password Manager(optional)

Using password managers such as Bitwarden or 1password means that you only have to remember one password instead of remembering a separate password for each site.
Just add your passwords to the manager and it will remember them for you!
Remember that not every password manager is good. I recommend using Bitwarden.
